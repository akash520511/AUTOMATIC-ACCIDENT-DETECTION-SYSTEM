## Features

### Core Detection
- **YOLOv8 Vehicle Detection** - Real-time vehicle detection with 95% mAP
- **Optical Flow Motion Analysis** - Track vehicle trajectories and speed
- **Collision Detection** - 30% overlap + 50% speed drop threshold
- **Severity Classification** - LOW (<65), MEDIUM (65-85), HIGH (>85)

### Advanced Features
- **Number Plate Recognition** - Automatic license plate detection
- **Victim Identification** - Detect persons and assess injuries
- **Family Notification** - Alert family members of involved vehicles
- **Emergency Alerts** - Simulate police, ambulance, fire dispatch
- **Heatmap Visualization** - Collision impact zone mapping
